noteseq.py¶

class common.noteseq.NoteSequencer(sched, synth, channel, program, notes, loop=True)¶

Plays a single Sequence of notes. The sequence is a python list containing notes. Each note is (dur, pitch).

Parameters

  • sched – The Scheduler object. Should keep track of ticks and allow commands to be scheduled.

  • synth – The Synthesizer object that will generate audio.

  • channel – The channel to use for playing audio.

  • program – A tuple (bank, preset). Allows an instrument to be specified.

  • notes – The sequence of notes to play, a list containing (dur, pitch).

  • loop – When True, restarts playback from the first note.

start()¶

Starts playback.

stop()¶

Stops playback.

toggle()¶

Toggles playback.
