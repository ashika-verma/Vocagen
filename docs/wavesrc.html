class common.wavesrc.WaveBuffer(filepath, start_frame, num_frames)¶

Reads certain data from a wave file and stores it in memory.

This is a WaveSource – a wave data providing interface. Call get_frames() to get audio data in the format we like (numpy array, float32).

Parameters

  • filepath – The path to the wave file. Should be a 16 bit file with a sample rate of 44100Hz.

  • start_frame – The frame of the wave file that this buffer should start on.

  • end_frame – The frame of the wave file that this buffer should end on.

get_frames(start_frame, end_frame)¶

Gets a range of frames of audio data from the wave buffer.

Parameters

  • start_frame – The frame of the wave buffer to start on.

  • end_frame – The frame of the wave buffer to end on.

Returns

A numpy array of audio data from start_frame to end_frame of the wave buffer. If more frames are asked for than are available, then returns what it can.

get_num_channels()¶
Returns

The number of channels of the loaded wave file.

class common.wavesrc.WaveFile(filepath)¶

Interface for reading data from a wave file. Does not store this data locally. Simply call get_frames() to get data in format we like (numpy array, float32).

Parameters

filepath – The path to the wave file. Should be a 16 bit file with a sample rate of 44100Hz.

get_frames(start_frame, end_frame)¶

Gets a range of frames of audio data from the provided wavefile.

Parameters

  • start_frame – The frame of the wave file to start on.

  • end_frame – The frame of the wave file to end on.

Returns

A numpy array of audio data from start_frame to end_frame of the wave file. If more frames are asked for than are available, then returns what it can.

common.wavesrc.make_wave_buffers(wave_path, regions_path)¶

Reads from a regions file and a wave file to create one WaveBuffer per region.

Parameters

  • wave_path – The path to the wave file.

  • regions_path – The path to the text file containing one region per line. Each line should contain the following, separated by tabs: start time (in seconds), index, length (in seconds), region name.

Returns

A dictionary of WaveBuffers, keyed by region name.
